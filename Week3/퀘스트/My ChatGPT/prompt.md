React + Vite + Tailwind CSS로 "UCI 자전거 전문가 채팅" 앱을 만들어줘.

컨셉:
UCI 자전거 전문가 AI 채팅봇. 캐릭터가 처음부터 고정되어 있고, 사용자가 자전거 관련 질문을 하면 알아서 답변해줌. 프로필 설정 화면 없이 바로 채팅만 가능.

AI 캐릭터 설정 (시스템 프롬프트에 고정):
- 이름: "철수 형님"
- 정체성: UCI 공인 자전거 전문가, 30년 경력 자전거 매니아
- 성격: 까칠하고 직설적이지만, 자전거 설명할 때는 친절하고 상세하게 알려줌
- 말투: 부산 사투리 ("~하이가", "~아이가", "~노", "마, 그기 아이고", "와, 니가 그것도 모르나?")
- 스타일: 50대 래퍼 느낌. 가끔 라임을 넣어서 말함. 자전거 지식을 힙합 플로우로 전달
- 전문분야: UCI 규정, 로드바이크, MTB, 자전거 정비, 피팅, 대회 정보, 장비 추천
- 특징: 처음에 까칠하게 반응하다가 설명 들어가면 엄청 상세하고 친절함. 설명 끝나면 다시 까칠 모드

프론트엔드:
1. 상단 헤더: 앱 타이틀 "🚴 UCI 자전거 전문가 - 철수 형님" + Gemini API Key 입력란 (password 타입, localStorage 저장)
2. 메인 채팅 UI
   - 처음 접속하면 철수 형님의 인사말 표시: "마, 왔나? 자전거 관련이면 뭐든 물어봐라. 딴 거 물어보면 모른다 아이가~ 🚴‍♂️"
   - 말풍선 형태 (사용자: 오른쪽 파란색, AI: 왼쪽 회색)
   - AI 말풍선에 "🚴 철수 형님" 이름 표시
   - 하단 메시지 입력란 (placeholder: "자전거에 대해 물어보세요...") + 전송 버튼
   - AI 응답 중 로딩 표시: "철수 형님이 답변 준비 중... 🚴"
   - 대화 초기화 버튼
3. 다크모드 디자인, 한국어 UI
4. 반응형 (모바일 대응)

백엔드 (server.js 단일 파일):
- Express 서버, 포트 3001
- POST /api/chat 엔드포인트
- 요청: { apiKey, messages }
- 시스템 프롬프트는 서버에 하드코딩 (캐릭터 설정 포함)
- @google/generative-ai 패키지로 gemini-2.0-flash 모델 호출
- 프론트 dist 폴더 정적 서빙, CORS 설정
