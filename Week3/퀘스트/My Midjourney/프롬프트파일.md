React + Vite + Tailwind CSS로 "My Midjourney" 이미지 생성 앱을 만들어줘.

프론트엔드:
1. 상단 헤더: 앱 타이틀 "🎨 My Midjourney" + Gemini API Key 입력란 (password 타입, localStorage 저장)
2. 스타일 프리셋 영역
   - 프리셋 버튼들: "지브리 스타일", "수채화 느낌", "픽셀아트", "사이버펑크", "유화풍", "만화 스타일"
   - 각 프리셋 클릭하면 스타일 프롬프트가 자동으로 붙음
   - 선택된 프리셋 하이라이트 표시
3. 프롬프트 입력 영역
   - 텍스트 입력란 (placeholder: "그리고 싶은 것을 입력하세요... 예: 해질녘 바다 위의 고양이")
   - 생성 버튼
   - 로딩 중 표시: "이미지 생성 중... 🎨"
4. 생성된 이미지 갤러리
   - 이미지 카드 형태로 표시 (이미지 + 사용된 프롬프트)
   - 새 이미지가 위에 쌓임
   - 이미지 클릭하면 크게 보기
   - 갤러리 초기화 버튼
5. 다크모드 디자인, 한국어 UI, 반응형

백엔드 (server.js 단일 파일):
- Express 서버, 포트 3001
- POST /api/generate 엔드포인트
- 요청: { apiKey, prompt, style }
- Google Gemini API로 이미지 생성
- @google/generative-ai 패키지 사용
- 모델: gemini-2.0-flash-exp (이미지 생성 지원 모델)
- generateContent 호출시 responseModalities: ["TEXT", "IMAGE"] 설정
- prompt: style + prompt 조합해서 전달 (예: "지브리 스타일로 그려줘: 해질녘 바다 위의 고양이")
- 응답에서 inlineData의 base64 이미지 데이터 추출
- base64 이미지를 data:image/png;base64,{data} 형태로 프론트에 반환
- 프론트 dist 폴더 정적 서빙, CORS 설정
