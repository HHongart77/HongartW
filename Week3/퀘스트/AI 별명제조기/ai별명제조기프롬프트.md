React + Vite + Tailwind CSS로 "AI 별명 생성기" 앱을 만들어줘.

프론트엔드:
1. 상단 헤더: 앱 타이틀 "🏷️ AI 별명 생성기" + Gemini API Key 입력란 (password 타입, localStorage 저장)

2. 사용자 정보 입력 폼
   - 이름 입력란 (placeholder: "이름을 입력하세요")
   - 성격 입력란 (placeholder: "예: 활발한, 조용한, 유머러스한...")
   - 취미 입력란 (placeholder: "예: 게임, 독서, 운동...")
   - 스타일 프리셋 버튼들: "귀여운 동물 이름 스타일", "게임 캐릭터 느낌", "힙한 영어 닉네임", "급식체 스타일", "판타지 용사 스타일"
   - 선택된 스타일 하이라이트 표시
   - "별명 생성하기" 버튼

3. 별명 결과 영역
   - AI가 생성한 별명 5개를 카드 형태로 표시
   - 각 카드에 별명 + 별명의 의미 설명
   - 각 카드에 "복사" 버튼 (클릭하면 클립보드에 복사, "복사됨!" 표시)
   - 각 카드에 "선택" 버튼 (클릭하면 하이라이트)
   - "다시 생성" 버튼
   - 로딩 중: "AI가 별명을 만들고 있어요... 🎲"

4. 다크모드 디자인, 한국어 UI, 반응형

백엔드 (server.js 단일 파일):
- Express 서버, 포트 3001
- POST /api/generate-nicknames 엔드포인트
- 요청: { apiKey, name, personality, hobby, style }
- @google/generative-ai 패키지 사용
- 모델: gemini-2.0-flash
- 시스템 프롬프트: "너는 재미있는 별명을 만들어주는 전문가야. 사용자 정보를 바탕으로 창의적이고 재미있는 별명 5개를 만들어줘. 각 별명에 왜 그 별명을 추천하는지 짧은 설명도 붙여줘. 반드시 JSON 배열로 응답해. 형식: [{ nickname: 별명, reason: 이유 }]"
- 사용자 프롬프트: "이름: {name}, 성격: {personality}, 취미: {hobby}, 스타일: {style}로 별명 5개 만들어줘"
- AI 응답에서 JSON 파싱해서 반환
- 프론트 dist 폴더 정적 서빙, CORS 설정
